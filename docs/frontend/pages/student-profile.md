# Page Spec — Student Profile

**Route:** `/student/profile`  
**Audience:** Student

## Objectives
- Capture academic and personal profile required for eligibility checks.

## Sections
1. Basic details (name, department, program, year)
2. Academic summary (CGPA, backlog count)
3. Skills and certifications pointers
4. Save/update actions

## Validation
- Client-side required-field checks.
- Inline errors and save success messages.
