# Page Spec — Student Drives

**Routes:**
- `/student/drives`
- `/student/drives/:driveId`

## Objectives
- Help students discover and apply to relevant drives.

## Listing Screen
- Filters (department, package range, location, employment type)
- Eligibility hint badge (Eligible / Check details)
- Deadline and status indicators

## Detail Screen
- Full eligibility criteria
- Drive timeline
- Apply action
- Opt-out guidance after apply
