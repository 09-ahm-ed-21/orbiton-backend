# Page Spec — Student Applications & Offers

**Routes:**
- `/student/applications`
- `/student/applications/:applicationId`
- `/student/offers`
- `/student/offers/:offerId`

## Objectives
- Provide transparent progress and decisions.

## Application Detail
- Status timeline (Applied -> Rounds -> Outcome)
- Round outcomes list
- Opt-out action with policy warning by stage

## Offer Detail
- Offer terms summary
- Decision actions: Accept / Decline
- Statuses: OFFERED, ACCEPTED, DECLINED, TIMED_OUT

## Escalations
- Post-offer opt-out triggers visible “Placement Office Follow-up” notice.
