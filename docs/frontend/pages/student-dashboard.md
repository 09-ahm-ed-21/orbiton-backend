# Page Spec — Student Dashboard

**Route:** `/student/dashboard`  
**Audience:** Student

## Objectives
- Give one-screen view of placement progress.
- Surface next actions quickly.

## Blocks
1. Profile completeness card
2. Resume status card
3. Applications summary (Applied / In Process / Rejected / Offered)
4. Upcoming rounds timeline
5. Recent notifications
6. Quick actions (Update profile, Upload resume, Browse drives)

## Key States
- First-time user with no profile
- Profile exists but no resume
- Active drive participation
