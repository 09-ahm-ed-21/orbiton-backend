# Page Spec — Landing Page

**Route:** `/`  
**Audience:** Public

## Objectives
- Explain Orbiton value proposition.
- Segment users into Student / Recruiter / TPO.
- Drive users to register/login.

## Sections
1. Hero with headline + CTA (`Get Started`, `Sign In`)
2. Problem/Solution summary
3. Role cards (Student, Recruiter, TPO)
4. Feature highlights (Drive management, Resume assistance, Tracking)
5. Trust strip (institution stats/demo metrics)
6. Footer with contact + support

## CTA Destinations
- `/auth/register`
- `/auth/login`

## Responsive Notes
- Stack cards vertically on `xs/sm`.
- Keep hero headline to 2–3 lines on mobile.
